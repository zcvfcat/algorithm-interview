# 빅오

## 관심 대상 `최고차항`

시간 복자도를 표현할 때 최고차항만 표기하며, 계수는 무시

입력값에 따른 실행 시간의 추이

## 빅오 표기법은 주어진 (최선 / 최악 / 평균) 경우의 수행 시간의 상한을 나타냄

### O(1)

상수값... 찾기 힘듬
11장 : 해시 테이블의 조회 및 삽입

### O(log n)

실행시간 입력값에 영향 받음
18장 : 이진 검색

### O(n)

정렬되지 않는 리스트에서 최대값 또는 최소값 찾는 경우

### O(n log n)

17장 병합정렬
비교 기간 정렬은 nlogn 을 넘기 힘듬

입력값이 최선의 경우 Timsort O(n)

### O(n^2)

17장 버블 정렬

### O(2^n)

23장 피보나치수
재귀로 계산하는 알고리즘

### O(n!)

12장 브루트 포스
